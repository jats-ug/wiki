# ビルド手順

## tar玉から

http://jats-ug.metasepi.org/#install を参照。

## gitリポジトリから

```
$ git clone https://github.com/githwxi/ATS-Postiats.git
$ cd ATS-Postiats
$ make -f codegen/Makefile_atslib
$ export PATSHOME=`pwd`
$ make -f Makefile_devl all
```

PATSHOMEを設定した状態で使うことになる。
